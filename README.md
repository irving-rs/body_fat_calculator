# body_fat_calculator
Body Fat Calculator: Estimates body fat, fat mass and lean mass.

Programmed in Python 3.8.5, uses CLI. 

## Details:
- Based on the US Navy formula.
- Metric unit system.
- Two models in one (Male and Female).
